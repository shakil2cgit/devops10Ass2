# DevOps Module 2 — Git & GitHub Assignment

## Project Description

This project is a hands-on assignment for learning **Version Control using Git and GitHub**. It covers essential Git concepts including repository setup, branching, merging, pull requests, and deployment workflows.

## Topics Covered

- Initializing a Git repository
- Staging and committing changes
- Connecting to a remote GitHub repository
- Creating and managing branches
- Pull Requests and code merging
- Deploying a web application
